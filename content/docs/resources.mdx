<p className="guide-intro">These tools are shared across every content type. Set them up once and they'll work everywhere.</p>

<div className="guide-card">
  <p><strong>Anki</strong> is a spaced-repetition flashcard app — it shows you words right before you're about to forget them. It's the single most effective tool for vocabulary retention and you'll use it throughout your learning.</p>
  <p><strong>Yomitan</strong> is a browser extension that lets you hover over any Japanese word to instantly get its reading, meaning, and pitch accent. It also integrates directly with Anki for one-click mining.</p>
  <p><strong>ShareX</strong> (PC only) handles screenshots and screen capture for mining — used alongside Yomitan to add images and audio to your Anki cards automatically.</p>
</div>

<span id="manga-tools" className="guide-anchor" />

#### Tools & Setup

<div className="guide-card">
  <p>Manga text is embedded in images, so you can't hover over it directly. There are two approaches:</p>
  <p><strong>Mokuro</strong> pre-processes manga pages so every speech bubble becomes selectable text — you can then hover with Yomitan as normal. This is the smoothest experience once set up.</p>
  <p><strong>OCR</strong> (via owocr + ShareX) lets you draw a box around any text on screen and have it scanned and sent to Yomitan's search page instantly. More flexible — works on any image, any app.</p>
  <p><strong>CDisplayEx</strong> is a clean manga reader for PC.</p>
</div>

<span id="vn-tools" className="guide-anchor" />

#### Tools & Setup

<div className="guide-card">
  <p>VNs require a <strong>texthooker</strong> setup — a tool that extracts the in-game dialogue and sends it to a browser page where Yomitan can read it. The standard tool is <strong>Textractor</strong>.</p>
  <p>The setup is: Textractor (hooks game text) → Texthooker page in browser (displays text) → Yomitan (hovers on words). Once running, it's seamless.</p>
  <p>You'll also need to set your <strong>system locale to Japanese</strong> for most VNs to run correctly on Windows.</p>
</div>

<Callout type="warn">
  Most Japanese VNs are Windows-only. If you're on Mac or Linux, see the Linux overview on Lazy Guide for options.
</Callout>
