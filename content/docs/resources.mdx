<span id="manga-tools" className="guide-anchor" />

#### Tools & Setup

<div className="guide-card">
  <p>Manga text is embedded in images, so you can't hover over it directly. There are two approaches:</p>
  <p><strong>Mokuro</strong> pre-processes manga pages so every speech bubble becomes selectable text — you can then hover with Yomitan as normal. This is the smoothest experience once set up.</p>
  <p><strong>OCR</strong> (Meikipop) lets you draw a box around any text on screen and have it scanned and sent to Yomitan's search page instantly. More flexible — works on any image, any app.</p>
  <p><strong>Manatan</strong> is an all-in-one app with automatic OCR — the easiest option for beginners, available on Desktop, iOS, and Android.</p>
  <p><strong>CDisplayEx</strong> is a clean manga reader for PC.</p>
</div>

<span id="vn-tools" className="guide-anchor" />

#### Tools & Setup

<div className="guide-card">
  <p>VNs require a <strong>texthooker</strong> setup — a tool that extracts the in-game dialogue and sends it to a browser page where Yomitan can read it. The standard tool is <strong>Textractor</strong>. A modern alternative is <strong>LunaTranslator</strong>.</p>
  <p>The setup is: Textractor (hooks game text) → Texthooker page in browser (displays text) → Yomitan (hovers on words). Once running, it's seamless.</p>
  <p>You'll also need to set your <strong>system locale to Japanese</strong> for most VNs to run correctly on Windows.</p>
</div>

<Callout type="warn">
  Most Japanese VNs are Windows-only. If you're on Mac or Linux, see the Linux overview on Lazy Guide for options.
</Callout>
